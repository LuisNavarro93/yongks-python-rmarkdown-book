18.8.3 Bag of Words

Tokenize each record, then convert into list of counters (words-frequency pair) Each item in the list is a counter, which represent word frequency within the record

bag_of_words = []
for text in movies.text:
    bag_of_words.append( Counter(
        casual_tokenize(text, reduce_len=True, strip_handles=True)
    ))
print("Total Rows:", len(bag_of_words))

18.8.4 Build The Model

from sklearn.naive_bayes import MultinomialNB
train_y  = movies.sentiment>0
train_X  = bows_df
nb_model = MultinomialNB().fit( train_X, train_y)

18.8.5 Train Set Prediction

train_predicted = nb_model.predict(bows_df)
